## Common run pattern

Each sample follows the same three-step flow:

<Steps>
  <Step title="Restore tools">
    ```bash
    dotnet tool restore
    ```
  </Step>
  <Step title="Generate the Bake plan">
    ```bash
    buildcharts generate
    ```
  </Step>
  <Step title="Run the build">
    ```bash
    docker buildx bake --file .buildcharts/docker-bake.hcl --no-cache
    ```
  </Step>
</Steps>
